# Stanford Memorial Church Inscriptions

An interactive visualization of the inscriptions found throughout Stanford Memorial Church. Built as part of the **AI for Human Flourishing** course at Stanford GSB (Winter 2026).

## About

Stanford Memorial Church features dozens of carved inscriptions — scriptural passages, spiritual reflections, and words of comfort — adorning its walls, arches, and mosaics. This project captures those inscriptions and presents them as an interactive word cloud.

## View

Open `Stanford Memorial Church Word Cloud.html` in any modern browser, or visit the live site via GitHub Pages.

## Files

- **Stanford Memorial Church Word Cloud.html** — Self-contained interactive visualization
- **Stanford Memorial Church Quotes.xlsx** — Source data of all collected inscriptions
